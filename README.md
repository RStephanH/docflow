# docflow

DocFlow is a full-stack document management platform designed to digitalize administrative processes.

## Overview

> DocFlow is a containerized full-stack application designed to demonstrate modern DevOps and software engineering practices.
> 
> The platform enables:
> 
> - Automated generation of administrative documents
> - Dynamic PDF creation
> - Simulated electronic signature workflow
> - Document monitoring dashboard
> - Containerized deployment with Docker
> - CI/CD pipeline with GitHub Actions
> 
> The project focuses on reliability, scalability, and infrastructure automation.
